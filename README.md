## Overview

This project is a small score-tracking interface built to practice front-end fundamentals and clear UI behavior.  
The focus is not on complexity, but on making interactions understandable and immediate for the user.

## What this project demonstrates

- Clear separation between user actions and displayed results
- Immediate visual feedback when scores change
- A simple, readable layout without unnecessary visual noise

## UX / UI considerations

- Actions are easy to distinguish and trigger
- Information is presented in a straightforward and scannable way
- The interface prioritizes clarity over decoration

- HTML
- CSS
- JavaScript

## Notes

This project was built as a fundamentals exercise.  
If expanded further, the next steps would include accessibility improvements, keyboard interaction support, and responsive refinement.
